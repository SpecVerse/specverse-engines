## Usage

These files can be used directly with:

### 1. Mermaid Live Editor
Copy the contents and paste into [mermaid.live](https://mermaid.live) for interactive viewing and editing.

### 2. GitHub Markdown
Include in GitHub README files:
```markdown
```mermaid
[paste contents here]
\```
```

### 3. Documentation Sites
- **Docusaurus**: Supports Mermaid diagrams natively
- **MkDocs**: Use `pymdown-extensions` for Mermaid support
- **GitBook**: Supports Mermaid via plugins

### 4. IDE Extensions
- **VSCode**: Install "Markdown Preview Mermaid Support" extension
- **IntelliJ**: Built-in Mermaid support in Markdown files

### 5. Image Export
Use Mermaid CLI to convert to PNG/SVG:
```bash
npm install -g @mermaid-js/mermaid-cli
mmdc -i 11-01-event-flow-sequence.mmd -o diagram.png
```

## Regenerating Diagrams

To regenerate any diagram:

```bash
# From the project root
cd examples/11-diagrams

# Regenerate a specific diagram
specverse gen diagram 11-01-event-flow-sequence-demo.specly -t event-flow-sequence -o generated-diagrams/11-01-event-flow-sequence.mmd

# Or regenerate all diagrams
for file in 11-0*.specly; do
  base=$(basename $file .specly)
  specverse gen diagram $file -o generated-diagrams/
done
```

## Notes

- These are the **complete, full-size diagrams** as generated by the SpecVerse diagram engine
- The `.md` documentation files contain **simplified versions** for readability
- All diagrams use the default theme - you can regenerate with different themes using the `-t` flag
- Diagrams are auto-generated and should not be manually edited - edit the `.specly` source instead
